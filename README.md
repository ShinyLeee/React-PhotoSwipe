# React-PhotoSwipe

一个面向移动端的纯React图片浏览组件， 受[PhotoSwipe](https://github.com/dimsemenov/PhotoSwipe)启发而编写。

## 与PhotoSwipe相比

- 支持放大、缩小经剪裁后的图片

- 支持以React组件的方式自定义UI-Template

- 面向移动端，体积更小

## 注意事项

React-PhotoSwipe暂不适用于生产环境
